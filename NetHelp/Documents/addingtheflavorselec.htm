Adding the Flavor Selector Module

Any entry is available in various transcoding flavors. The flavor selector module allows you to manually select the video flavor you want to watch, based on your preferences.  The options are HD on or HD off.
If you are broadcasting with Adaptive RTMP then the flavor selector will be set to “auto”.

  To add the Flavor’s Selector module

1.   Go to Studio tab and create a new player or edit an existing one.

2.   In the Features tab, go to the Features List and click Controls.

3.   Check Flavor Selector.

4.   Select Options (optional) to configure display options and Save Changes.

5.   Click Preview to view your selection and click Save Changes to complete the process.
