Configuring VAST Ads

  To configure VAST Ads

1.   Select the Studio tab and select an existing player or create a new one. See Designing and Configuring a Player. The player is configured to display VAST ads only.

2.   Select the Advertising tab.

3.   Enable Request ads for this player.

4.   Configure the Ad Source as VAST Ad server.
