Edit - Remove Categories

You can remove categories from entries in the KMC. Removal will propagate to other applications.

  To remove categories  from an entry

1.   Select the Content tab and then select the Entries tab.

2.   Select an entry or multiple entries and select Bulk Actions on the bottom of the page.

3.   Select Remove Categories from the drop-down menu.

4.   Check the categories to remove from the all selected entries.

5.   Click Apply.
