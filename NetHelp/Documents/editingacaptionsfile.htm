Editing a Captions File

After you upload a captions file you can edit the file, remove it, or set it as the default captions file.

  To edit a captions file

1.   Select the Content tab and click on an entry.

2.   Select the Captions tab.

3.   In the Actions column, click Download File.

4.   Edit the file and then upload the file again as described in Adding Captions to an Entry.

5.   Select Remove to delete the captions file.
