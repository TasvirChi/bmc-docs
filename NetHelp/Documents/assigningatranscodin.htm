Assigning a Transcoding Profile to a Bulk Upload

A default Transcoding profile is included with the KMC. After you create a Transcoding Profile, an ID is created and listed in the Transcoding Profiles page.

  To assign a transcoding profile other than the default profile, to a bulk upload

•      Set the ID from the Transcoding Profiles to the conversionProfileId in your CSV.

•      Set the ID from the Transcoding Profiles List to the conversionProfileId in your XML file.
